# 📊 Matplotlib Complete Guide & Mini Project 🚀

This repository documents my learning journey of **Matplotlib**, a powerful Python library used for data visualization. It also includes a mini project demonstrating real-world data analysis and visualization.

---

## 📌 About Matplotlib

Matplotlib is one of the most widely used Python libraries for creating static, animated, and interactive visualizations. It is essential for Data Science, Machine Learning, and data analysis workflows.

## 🚀 Mini Project: Student Performance Analysis

### 🎯 Objective

Analyze student academic performance and visualize it using different types of plots.

### 📁 Project Structure

```
student-performance-analysis/
│── data.csv
│── analysis.py
│── README.md
```

### 📊 Features

* Total marks calculation
* Average marks analysis
* Multiple visualizations:

  * Bar Chart (Total Marks)
  * Line Plot (Average Marks)
  * Pie Chart (Subject Distribution)
  * Histogram (Marks Distribution)
  * Scatter Plot (Subject Comparison)
